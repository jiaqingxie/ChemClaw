## Install

```bash
npx skills add InternScience/ChemClaw
```

Install only one skill:

```bash
npx skills add InternScience/ChemClaw --skill mol-3d-viewer
```

List skills from this package:

```bash
npx skills add InternScience/ChemClaw --list
```
