## Install

```bash
npx skills add AI4Chem/ChemClaw
```

Install only one skill:

```bash
npx skills add AI4Chem/ChemClaw --skill mol-3d-viewer
```

List skills from this package:

```bash
npx skills add AI4Chem/ChemClaw --list
```
